# sudoenvx portfolio

A clean, modern personal portfolio built with Next.js. It showcases selected work, a concise bio, and an easy way to get in touch.

## Highlights
- Fast, responsive layout optimized for mobile and desktop
- Simple structure focused on content and clarity
- Contact page for direct outreach

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- CSS

## Getting Started
```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Project Structure
- `app/` – App Router pages and layouts
- `public/` – Static assets
- `styles/` – Global styles (if applicable)

## Customize
- Update content in the `app/` routes
- Replace assets in `public/`
- Tweak styling to match your personal brand
